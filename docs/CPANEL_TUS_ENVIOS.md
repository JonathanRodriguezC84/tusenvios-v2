# Despliegue en cPanel - Tus Envios

Proyecto para `tusenvios.com.co`.

## Requisitos

- PHP 8.3
- MySQL
- Composer
- Node solo para compilar assets localmente
- Acceso SSH recomendado

## Variables principales

```env
APP_NAME="Tus Envios"
APP_URL=https://tusenvios.com.co
DB_DATABASE=tus_envios
```

## Recordatorio

El proyecto separado debe estar en:

```text
C:\laragon\www\tus-envios
```

No desplegar desde `rci-envios`, porque esa carpeta queda como proyecto base/original.
